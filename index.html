Learn Subnetting Simply

üåê What is an IP Address?

An IP address is like a house address for computers.

Example: 192.168.1.10

192 City
168 Street
1 Block
10 House #

üèòÔ∏è What is a Subnet?

A subnet is a group of computers that can talk directly.

Think of it as a neighborhood:

Downtown (192.168.1)
192.168.1.1 192.168.1.10 192.168.1.100 192.168.1.254
Uptown (192.168.2)
192.168.2.1 192.168.2.10 192.168.2.100 192.168.2.254

If two IP addresses have the SAME first 3 numbers,
they're in the SAME subnet!
